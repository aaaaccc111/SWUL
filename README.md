# 軟體使用紀錄系統

隨著資訊技術的快速發展，現代企業和個人越來越依賴於各類應用程式和軟體工具，以提升工作效率並確保紀錄的安全與合法性。然而，軟體的使用情況和資源消耗常常難以有效追蹤，導致資源浪費或潛在的問題未能及時發現。在此需求下，設計了一套軟體使用紀錄系統，可達成使用行為透明化、資源使用監控、紀錄整合與分析。
透過這套系統，可以迅速且方便的看見軟體使用情況，減少不必要的資源浪費、提高作業效率、強化紀錄的透明度、有效的給予需求單位管理軟體使用方式。  

前端 :  
* HTML5 / CSS3 / JavaScript / Chart.js  
  
後端 :  
* Python Flask 框架  
  
Client 端應用程式  
* Python

系統運作模式及介紹  
* 採用 C/S 分離式架構， Client 端與 Server 個別運作模式，確保即便網路異常或 Server 端維護，終端紀錄亦不受影響  
* 採用 Python WNI 架構做開發，達成抓取電腦資訊方法，並且穩定於電腦背景做執行(不影響原先使用者使用)  
* 透過 Chart.js 實現動態圖表，將滿是數字的的 log 轉化為直觀的軟體使用趨勢與效率報表  
* Client 端資料可上傳至 Server 由 Server 做彙整，並且支援離線作業  
* 啟動時自動比對 Server 端版本號進行熱更新，確保環境一致性並降低維護成本  
* 支援軟體清單自定義過濾，可針對特定高單價或關鍵軟體進行重點紀錄
